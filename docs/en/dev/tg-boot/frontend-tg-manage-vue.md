# tg-manage-vue

Sample **admin console** shipped inside the TG-boot repo: Vue 3, TypeScript, Element Plus, Vue Router, Pinia, Axios, i18n, and Vite—aligned with the backend starters for demos.

## Local run

```bash
cd tg-manage-vue
npm install
npm run dev
```

Production build:

```bash
npm run build
```
